Ollama setup

OLLAMA_API_KEY Your Ollama API key
OLLAMA_MODEL gpt-oss:120b
OLLAMA_BASE_URL Only for your own server
  1. Deploy Astra_AI to Vercel from GitHub.
  2. Open Project Settings, then Environment Variables.
  3. Add OLLAMA_API_KEY with your Ollama API key.
  4. Add OLLAMA_MODEL with gpt-oss:120b for Ollama Cloud.
  5. Add OLLAMA_BASE_URL only if you use your own Ollama server.
  6. Redeploy the app, then choose a model in AI chatbot.

Project setup

  1. Create a project in the Project section.
  2. Choose that project in the AI chatbot project selector.
  3. Every reply will use the active project name and goal.
  4. For faster project replies, use Pro and a stronger Ollama model.
